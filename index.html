Mi primera vez en git

git init

Sirve para inicializar el repo


git status

Sirve para revisar los archivos faltantes o archivos dentro del repo


git add .

Sirve para incluir todos los archivos al repo


git commit -m "mensaje que quieras"

Sirve para describir los cambiso que hiciste de manera general


git remote add origin nombre_del_repo

Sirve para conectarse al repositorio remoto (el repo de la pagina de git)


git branch -M main

Sirve para renombrar de master a main que es la rama por defecto de git


git push -u origin nombre_de_la_rama

Sirve para subir los cambios a la rama remota mencionada
